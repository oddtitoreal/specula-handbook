## Repository GitHub

L'ecosistema Specula è distribuito su tre repository pubblici:

| Repository | Ruolo |
|---|---|
| [`specula-framework`](https://github.com/oddtitoreal/specula-framework) | Runtime specification layer — schemi JSON, contratti di fase, validazione eseguibile. Versione attuale: 1.2.0 |
| [`specula-method`](https://github.com/oddtitoreal/specula-method) | Methodology layer — il processo in sei fasi, framework etico, logica decisionale |
| [`specula-skill`](https://github.com/oddtitoreal/specula-skill) | Implementation layer — validazione della costituzione, state-machine, mapping constitution-to-runtime |

Questo manuale è il layer di documentazione operativa: spiega il metodo in forma leggibile e citabile, senza dipendere dalla narrazione del libro né dalla formalizzazione tecnica del framework.
