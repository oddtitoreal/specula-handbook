# Security Policy

Questo repository contiene documentazione, non codice eseguibile. Le segnalazioni di sicurezza rilevanti riguardano:

- contenuti che espongono dati personali o credenziali;
- link a destinazioni compromesse o ingannevoli;
- problemi nella supply chain di pubblicazione (configurazione Mintlify, GitHub Actions se presenti).

## Come segnalare

Non aprire una issue pubblica per problemi di sicurezza. Scrivi a **tito@marcolivi.it** descrivendo il problema e dove si trova. Riceverai risposta entro 7 giorni.

Per vulnerabilità nel codice dell'ecosistema (runtime, validatori), usa le security policy dei repo [`specula-framework`](https://github.com/oddtitoreal/specula-framework/security/policy) e [`specula-method`](https://github.com/oddtitoreal/specula-method/security/policy).
